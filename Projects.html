Thunkable Project

September 24, 2018

 

I dragged some blocks to create the timer stop system and values, and I had a timer variable and a score variable, once the timer stopped, the next block would display the value of the score. The difficulties I had with my code was that the app only uses 3mb of ram, so i had to optimize several blocks and making shorter and simpler blocks to get it to run at a slower pace without degrading the time that much. The other difficulty that I had was that the timer would work, but not display time, so I made a loop that displays the time everytime the timer updates. This was an independent project.
