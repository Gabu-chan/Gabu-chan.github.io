Repl.it Short Story Project

This is a collaborative project. We started by making the intro and putting it in a loop, typed some stuff here and there, and then we reached my final product. We had to do some research in order to complete the project, we had to have it get through a special set of requirements that we wanted, and we had to organize our team so we could collaborate efficiently. We had many difficulties, the first one is the looping issue, if we did not code it right, it would crash the server, so we decided to put a wait command so it does not crash. The second was getting the program to remember the decisions picked so it could change other parts of the story.
